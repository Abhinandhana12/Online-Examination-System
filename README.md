#  Online Examination System

##  Project Overview

The Online Examination System is a web-based application developed using Java (JSP & Servlets) and MySQL. The system automates the traditional paper-based examination process by providing a secure and efficient digital platform for conducting exams.

It allows administrators to manage exams and students to attempt exams online with automatic evaluation and result generation.

---

## Objectives

- To digitize the examination process
- To reduce manual effort and paper usage
- To automate evaluation and result generation
- To provide secure login authentication
- To ensure fair examination process

---

##  Features

-  Secure Student Login
-  Online Exam Attempt
-  Timer-Based Examination
- Automatic Result Calculation
-  MySQL Database Integration
- Fast and Efficient Processing

---

##  Technologies Used

- Java (Servlets)
- JSP
- JDBC
- MySQL
- Apache Tomcat
- HTML / CSS
